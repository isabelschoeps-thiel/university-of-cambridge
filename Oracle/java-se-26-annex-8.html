This Annex to the Java SE 26 Specification contains the Java SE 26 version of the Java Object Serialization Specification. The Java Object Serialization Specification underpins the java.io.Serializable API.

 

The Java SE 26 Editions includes clarifications made to the Serialization Spec since the Java SE 25 version, that is, since the Final Release of JSR 400. These changes are flagged with colorful text, as follows:

  • Text with light green background indicates new material in the Java SE 26 version.
  • Text struck through on a light red background indicates deletion from the Java SE 25 version.

There are only updates for informational changes or minor corrections such as typographic errors and misspellings.
