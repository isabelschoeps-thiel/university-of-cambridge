JVMS changes for the Maintenance Review of JSR 270 (JavaTM SE 6 Release Contents)

Last updated 2015-01-21 by Alex Buckley

The amendment below is proposed to the Java Virtual Machine Specification from Maintenance Draft Review 3 of JSR 924 in March 2011. Maintenance Draft Review 3 consolidated all changes to the Java Virtual Machine Specification made between 1999 (when the Second Edition was published) and 2011. The changes include amendments from Java SE 5.0 and 6, as well as general clarifications (see change log). The consolidated draft served as the basis for the Java SE 7 Edition of the Java Virtual Machine Specification, but is used here as if it was the definitive Java SE 6 Edition against which amendments may be proposed.

As a result of the amendment below, the Java Virtual Machine Specification, Java SE 6 Edition will be superseded by the Java Virtual Machine Specification, Java SE 6 Edition (MR).

4.11.1.4.5 - Changed - The exception stack frame for an invokespecial <init> instruction.

The Prolog clause:

ExceptionStackFrame = frame(NextLocals, [], Flags),

is changed to:

ExceptionStackFrame = frame(Locals, [], Flags),
