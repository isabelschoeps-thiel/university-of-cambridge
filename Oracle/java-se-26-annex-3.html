This Annex to the Java SE 26 Specification contains the Java SE 26 Editions of The Java Language Specification (JLS) and The Java Virtual Machine Specification (JVMS).

 

The Java SE 26 Editions include corrections and clarifications made to the JLS and JVMS since the Java SE 25 Editions, that is, since the Final Release of JSR 400. These changes are flagged with vertical bars and colorful text, as follows:

  • Blue text indicates a modification of material in the Java SE 25 Edition.
  • Red text indicates new material in the Java SE 26 Edition.

 

The Java SE 26 Platform includes one preview feature whose specification is logically part of the Java SE 26 Edition of the JLS, but is physically distributed in a standalone document:

  • JEP 530 Primitive Types in Patterns, instanceof, and switch (Fourth Preview)
