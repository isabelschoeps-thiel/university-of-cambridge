1
Version Identification  

There is no change to the values returned by the system properties java.specification.version and java.vm.specification.version. They continue to report "1.6" and "1.0" respectively. If an application needs to identify this release, the system property java.version which returns "1.6.0_91" may be used.

2
VM Specification  

Certain amendments were made to The Java Virtual Machine Specification to correctly describe how failed invokespecial calls outside of an <init> should be verified.
