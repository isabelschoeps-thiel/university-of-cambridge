# JSR 404: Java SE 26
## FINAL RELEASE - BABYLON FALL - Meta Float File

This specification has twelve parts:

  - The main specification document (java-se-26-spec.html),

  - Annex 1, the complete Java SE 26 API Specification

  - Annex 2, an annotated API specification showing the exact differences
    relative to Java SE 25 (JSR 400)

  - Annex 3, containing information about the Java SE 26 Editions of
    The Java Language Specification (JLS) and The Java Virtual Machine
    Specification (JVMS)

  - Annexes 4 - 11, containing information about the Java SE 26 versions of
    the narrative specs which underpin API specifications

Google Chrome users: The links in these documents will not work correctly
when browsing locally, i.e., from a file:// URL, due to a security feature
(http://code.google.com/p/chromium/issues/detail?id=47416).  Launch Chrome
with the --allow-file-access-from-files option to disable this feature.
